## Rogue Squad <span class="not-bold">— *Indie Video Game*</span>
Rogue Squad is a multiplayer cooperative roguelike built within the Godot Engine. This project makes heavy use of custom netcode. My YouTube devlogs on this project have reached up to 5k views.

-----

## Vector Engine <span class="not-bold">— *Procedural Animation Engine*</span>
Vector Engine is a tool I’ve developed to help me create animations for my YouTube channel. Used in 9 videos, it utilizes Rust, Vulkan, and Tauri to render animations defined procedurally through TypeScript scripts.

-----

## Mew Compiler <span class="not-bold">— *Custom Programming Language*</span>
The Mew Compiler is a side project I’ve been working on to test my capabilities of building a fully featured programming language. The Mew Compiler compiles high-level text based code down to Scratch blocks.
